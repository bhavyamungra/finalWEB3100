JavaScript-Based Weather Application

This JavaScript-based weather application is a dynamic and interactive tool that provides real-time weather updates for cities worldwide. By leveraging a reliable weather API, the app fetches accurate data and displays essential weather details, including temperature, humidity, wind speed, and a corresponding weather icon (e.g., sunny, rainy, or cloudy) that reflects current conditions.

Number Guessing Game

This Number Guessing Game is a simple yet engaging Python-based application that challenges players to guess a randomly generated number within a predefined range. The game provides feedback after each guess, indicating whether the guessed number is too high, too low, or correct. This makes the game both entertaining and a great exercise in logical thinking and problem-solving.
